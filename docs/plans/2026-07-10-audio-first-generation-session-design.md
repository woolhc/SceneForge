# Audio-First Generation Session

一键生成统一为音频优先流水线：文案输入先生成完整 Fish Audio 旁白，导入音频直接标准化为主旁白；两种入口随后共享一次 Whisper 转写、AI 分镜富化、素材评分选择、时间线构建和字幕生成。

`GenerationSession` 是可持久化的阶段状态，保存项目、来源、旁白、转写、分镜、素材选择、错误和生成报告。时间轴只以主旁白真实时间戳为准。字幕直接从同一份 transcript 创建，不再重复执行 ASR。

素材选择使用可测试的评分器，综合画幅、时长、语义、素材方向、搜索排名和重复惩罚。低于置信度阈值时不强行绑定，保留候选并在报告中标记为建议手动选择/文字卡。
